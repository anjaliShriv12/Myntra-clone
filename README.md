# Myntra Clone

A responsive frontend clone of the Myntra homepage built using HTML and CSS. This project focuses on recreating the layout, styling, and responsive design of a modern e-commerce interface.

## Features

* Responsive homepage layout
* Navigation bar with categories
* Search bar design
* Banner sections and promotional offers
* Product category sections
* Footer section similar to Myntra

## Tech Stack

* HTML
* CSS

## Folder Structure

```text
/project-root
│── category/
│── offers/
│── banner.jpg
│── banner2.jpg.gif
│── index.html
│── style.css
│── utils.css
│── myntralogo.jpg
```

## Installation

1. Clone the repository

```bash
git clone https://github.com/anjaliShriv12/Myntra-clone.git
```

2. Open project folder

```bash
cd Myntra-clone
```

3. Run the project

Open `index.html` in browser
